Create Experiment Class with Constructor and Destructor 

   A constructor that prints "Constructor called for object X" (where X is an ID).  
   A destructor that prints "Destructor called for object X".  

  In the function objectArray():  
   Declare an array of 3 objects with different IDs (e.g., 1, 2, 3).  
   Print "Array of objects created".  

  In the main() function:  
   Call objectArray().  
   Observe the order of constructor and destructor execution.
