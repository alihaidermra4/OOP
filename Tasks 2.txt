1. Create a "Student" class
Write a C++ program that defines a class Student with the following private data members:

name (character array of size 50)
rollNumber (integer)
marks (float)

Provide setter functions for all the attributes and a display function to print the student's details.
In main(), create an object of the class, set values, and display the student’s information.


2. Create a "BankAccount" class
Write a program that includes a BankAccount class with:

Private members: accountNumber (integer), balance (float)
Public methods:
setAccount() to set account details
deposit() to add money to the balance
withdraw() to subtract money (ensure balance doesn’t go negative)
display() to print account details
Create an object of BankAccount, set account details, perform deposit and withdrawal, and display the final balance.
